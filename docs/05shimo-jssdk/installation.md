# 石墨 JS-SDK

以下提供两种下载方式：

- 通过github下载：https://github.com/shimo-open/shimo-js-sdk/releases
- 通过 npm 等包管理工具安装：

```json
# 通过 npm
npm install --save shimo-js-sdk

# 通过 yarn
yarn add shimo-js-sdk
```
